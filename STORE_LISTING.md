# Chrome ウェブストア掲載情報

Chrome ウェブストアへの申請時に必要な情報のドラフトです。そのままコピーして使用できます。

## 説明 (Description)

Tab Capture Studio は、ブラウザのタブ画面の録画、スクリーンショット撮影、およびウィンドウサイズの調整を行うための Chrome 拡張機能です。

【主な機能と仕様】

■ 画面録画
アクティブなタブをワンクリックで録画開始します。

・音声録音: タブ内部の音声およびマイク音声の録音に対応しています。
・画質設定: フレームレート（最大 60fps）やビットレートの調整が可能です。
・カウントダウン: 録画開始前にカウントダウン（0, 3, 5, 10 秒）を設定できます。
・出力形式: WebM 形式で保存されます。

■ スクリーンショット
表示中のタブを画像として保存します。

・対応形式: PNG, JPEG, WebP から選択できます。
・品質設定: 保存時の画質を調整できます。

■ ウィンドウサイズ調整
ブラウザのウィンドウサイズを、プリセットされた解像度へ変更します。

・主要なアスペクト比（16:9, 16:10, 4:3）に対応した解像度を選択可能です。
・具体的な解像度例: 1280x720 (720p), 1920x1080 (1080p), 1024x768 など。

■ データの取り扱いについて
本拡張機能で録画した動画や撮影したスクリーンショットは、すべてユーザーのローカル環境（ブラウザ内および PC 内）で処理・完結します。データが外部サーバーへ送信されることはありません。

---

## 単一用途の説明 (Single Purpose Description)

この拡張機能は、ワンクリックで現在開いているタブの画面録画およびスクリーンショット撮影を行うことを主な目的としています。また、特定の解像度（1280x720 など）へウィンドウサイズを調整する補助機能も提供します。

## 権限が必要な理由 (Justification for Permissions)

### activeTab が必要な理由

ユーザーが拡張機能のアイコンをクリックしたりショートカットキーを使用した際に、現在アクティブなタブに対して画面録画やスクリーンショット、ウィンドウサイズの変更を行うために必要です。これにより、すべてのサイトへの常時アクセス権限を要求することなく、必要な時だけ機能を提供します。

### scripting が必要な理由

以下の機能を実現するために、ページ内に一時的なスクリプトを実行する必要があります。

・カウントダウン表示: 録画開始前に、ページ上にカウントダウンオーバーレイを表示するため。
・ウィンドウサイズの計算: ウィンドウリサイズ機能において、現在の正確なビューポートサイズを取得し、フレームサイズを含めた適切なウィンドウサイズを計算するため。

### tabCapture が必要な理由

この拡張機能の核となる機能である「タブの画面録画」を行うために必要です。対象タブの映像ストリームを取得します。

### downloads が必要な理由

録画した動画ファイルおよび撮影したスクリーンショットを、ユーザーのローカル環境に保存するために必要です。

### offscreen が必要な理由

Service Worker 内では一部の API が制限されているため、オフスクリーンドキュメントを使用してバックグラウンドで録画データの処理や音声の合成を行うために使用しています。

### storage が必要な理由

ユーザーの設定（録画のフレームレート、ビットレート、マイクの ON/OFF、種類、前回使用したウィンドウサイズなど）を保存し、次回起動時に復元するために必要です。
