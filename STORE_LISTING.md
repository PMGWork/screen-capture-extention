# Chrome ウェブストア掲載情報

Chrome ウェブストアへの申請時に必要な情報のドラフトです。そのままコピーして使用できます。

## 説明 (Description)

**Tab Capture Studio** は、ブラウザのタブ画面の録画とスクリーンショット撮影を、誰でも簡単に行える Chrome 拡張機能です。
複雑な設定は不要。インストールしてすぐに、高品質な画面収録を開始できます。

### 【主な特徴】

🎥 **きれいな画面録画**
アクティブなタブをワンクリックで録画開始。

- **音声録音**: タブ内の音声はもちろん、マイク音声の同時録音にも対応。解説動画の作成に最適です。
- **画質調整**: PC のスペックや用途に合わせて、フレームレート（最大 60fps）やビットレートを調整可能。
- **カウントダウン**: 録画開始前のカウントダウン（3 秒/5 秒/10 秒）で、準備する余裕を持てます。

📷 **高機能スクリーンショット**
表示されているタブの内容を瞬時に画像として保存。

- **選べるフォーマット**: PNG, JPEG, WebP から用途に合わせて選択可能。
- **品質設定**: ファイルサイズを抑えたい場合は品質を調整できます。

📏 **ウィンドウサイズ調整（開発者・配信者向け）**
「1280x720 (HD)」や「1920x1080 (FHD)」など、一般的な解像度へワンクリックでウィンドウサイズを変更できます。

- 録画範囲を固定したい場合や、レスポンシブデザインの確認にも便利です。
- 16:9, 16:10, 4:3 の各種アスペクト比に対応。

### 【こんな方におすすめ】

- 操作マニュアルやデモ動画を作成したい方
- ゲームプレイやウェブサイトの挙動を録画したい方
- バグ報告のために画面を記録したい開発者
- プレゼンテーションの練習を記録したい方

### 【プライバシーとセキュリティ】

録画データやスクリーンショットは、すべてお使いの PC 内で処理・保存されます。
外部のサーバーに映像が送信されることはありませんので、安心してご利用いただけます。

## 単一用途の説明 (Single Purpose Description)

この拡張機能は、ワンクリックで現在開いているタブの画面録画およびスクリーンショット撮影を行うことを主な目的としています。また、開発者向けに特定の解像度（1280x720 など）へウィンドウサイズを調整する補助機能も提供します。

## 権限が必要な理由 (Justification for Permissions)

### activeTab が必要な理由

ユーザーが拡張機能のアイコンをクリックしたりショートカットキーを使用した際に、現在アクティブなタブに対して画面録画やスクリーンショット、ウィンドウサイズの変更を行うために必要です。これにより、すべてのサイトへの常時アクセス権限を要求することなく、必要な時だけ機能を提供します。

### scripting が必要な理由

以下の機能を実現するために、ページ内に一時的なスクリプトを実行する必要があります：

1. **カウントダウン表示**: 録画開始前に、ページ上に「3, 2, 1」というカウントダウンオーバーレイを表示するため。
2. **ウィンドウサイズの計算**: ウィンドウリサイズ機能において、現在の正確なビューポートサイズ（`innerWidth`/`innerHeight`）を取得し、フレームサイズを含めた適切なウィンドウサイズを計算するため。

### tabCapture が必要な理由

この拡張機能の核となる機能である「タブの画面録画」を行うために必要です。`getMediaStreamId` を使用して、対象タブの映像ストリームを取得します。

### downloads が必要な理由

録画した動画ファイル（WebM など）および撮影したスクリーンショット（PNG/JPEG/WebP）を、ユーザーのローカル環境に保存するために必要です。

### offscreen が必要な理由

Manifest V3 の制限により、Service Worker 内では DOM 操作や MediaRecorder API が完全には使用できないため、オフスクリーンキュメントを作成してバックグラウンドで録画データの処理や音声の合成を行うために使用しています。

### storage が必要な理由

ユーザーの設定（録画のフレームレート、ビットレート、マイクの ON/OFF、スクリーンショットのフォーマット、前回使用したウィンドウサイズなど）を保存し、次回起動時に復元するために必要です。
